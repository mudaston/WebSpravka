## Run Locally

Clone the project

```bash
  git clone https://github.com/mudaston/WebSpravka.git
```

Go to the project directory

```bash
  cd WebSpravka
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  gulp
```

Open in a browser

```bash
  http://localhost:3000
```
